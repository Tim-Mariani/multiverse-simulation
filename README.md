# multiverse-simulation

6 adjustable parameters (gravity, entropy, expansion rate, exotic matter density, quantum uncertainty, founding civilization count)

turn-based "Advance Turn" button plus a real-time toggle (pause / 1x / 3x / 8x), with emergent interactions each tick — economy funds tech, tech and stability drive population growth, gravity/entropy drive stability drift and event frequency, and civilizations form trade routes, alliances, or wars with each other based on proximity in the stat space.

Supernova, Extinction, AI Singularity (with a chance of going rogue and declaring war on everyone), Golden Age, First Contact, and Meteor Storm — all manually triggerable.

an animated orbital "Universe Map" (starfield, pulsing central star, civilizations as orbiting nodes connected by color-coded diplomacy lines) plus canvas-drawn line/bar charts for population, civilization count, resource distribution, and average tech level over time.

click any civilization to see population, resources, stability/aggression bars, a 7-stage tech tree, and live diplomatic relations.

a near-invisible dot in the bottom-right corner opens a developer console to override physics constants directly, spawn or delete civilizations, or edit the entire universe state as raw JSON.
